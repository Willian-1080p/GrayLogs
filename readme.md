# 🛡️ Projetos de Monitoramento e Segurança - WillSecLab

Bem-vindo ao meu laboratório pessoal de **monitoramento e segurança da informação**.  
Este repositório reúne projetos práticos voltados para **centralização de logs, detecção de eventos suspeitos e uso de inteligência artificial aplicada à segurança (SIEM)**.

---

## 📁 Estrutura do Lab

- `graylog/` – Stack de centralização de logs com Docker (MongoDB + OpenSearch + Graylog)  
- `winlogbeat/` – Coleta de logs Windows e envio para o Graylog  
- `graylog-llm/` – Integração do Graylog com LLM para análise inteligente de logs  
- `scripts/` – Scripts auxiliares para automação e análise  

---

## ⚙️ Tecnologias/Ferramentas

- Graylog (SIEM / Log Management)  
- Winlogbeat (coleta de logs Windows)  
- Ollama (LLM local para análise de logs)  
- OpenSearch (indexação e busca)  
- MongoDB (armazenamento de configuração)  
- Python (integração e automação)  
- Docker & Docker Compose  

---

## 🧠 Conceitos Aplicados

- SIEM (Security Information and Event Management)  
- Centralização e correlação de logs  
- Monitoramento de eventos de segurança  
- Detecção de anomalias

Desenvolver um ambiente prático de **monitoramento e análise de segurança**, simulando cenários reais de infraestrutura corporativa, com foco em:

- 🚨 Detecção de possíveis incidentes de segurança  
- 🤖 Uso de IA (LLM) para análise automatizada de logs
